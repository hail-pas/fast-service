# fast-service
micro service based on FastApi

# 模块
## conf 配置模块
## core 核心模块
## deploy 部署模块
## entrypoint 启动模块
## scripts 脚本模块
## storages 存储模块
## apis 功能模块
    - graphql
    - http
        - version control
        - sub app
    - websocket
    - grpc
## third_apis 外部服务模块
## common 通用模块
## tests 测试模块
## tasks k8s job 和 cronjob 模块

# Pre-Commit
```shell
# 格式化
make style
# 代码检查
make check
# 测试用例
make test
```
